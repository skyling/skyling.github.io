Redis的安装以及简单使用

发表于 2015-10-14 | 分类于 linux | 阅读次数

前言

Redis是一个key-value存储系统。和Memcached类似，它支持存储的value类型相对更多，包括string(字符串)、list(链表)、set(集合)、zset(sorted set –有序集合)和hash（哈希类型）。这些数据类型都支持push/pop、add/remove及取交集并集和差集及更丰富的操作，而且这些操作都是原子性的。在此基础上，redis支持各种不同方式的排序。与memcached一样，为了保证效率，数据都是缓存在内存中。区别的是redis会周期性的把更新的数据写入磁盘或者把修改操作写入追加的记录文件，并且在此基础上实现了master-slave(主从)同步。

在Windows中的安装Redis

在Windows系统中安装Redis是件特别简单的事情，只需要下载可执行文件，然后按照步骤安装即可。下载地址：https://github.com/rgl/redis/downloads

这里需要按照你的计算机的系统的来下载，你的系统是32位的就下载32位的，64就下在64位的程序，这个很清楚。

安装好后运行服务端和客户端,我们的操作都是在客户端完成的。

Redis服务端：Redis目录/redis-server.exe

Redis客户端：Redis目录/redis-cli.exe
