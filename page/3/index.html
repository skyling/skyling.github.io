发表于 2015-11-13 | 分类于 php | 阅读次数

PHP接口是什么？

接口是两个PHP对象之间的契约，其目的不是让一个对象依赖另一个对象的身份，而是依赖另一个对象的能力。接口把我们的代码和依赖解耦了，而且允许我们的代码依赖任何实现了预期结果的第三方代码。我们不管第三方代码时如何实现接口的，只关心第三方代码是否实现了指定的接口。
