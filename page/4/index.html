Redis的安装以及简单使用

发表于 2015-10-14 | 分类于 linux | 阅读次数

前言

Redis是一个key-value存储系统。和Memcached类似，它支持存储的value类型相对更多，包括string(字符串)、list(链表)、set(集合)、zset(sorted set –有序集合)和hash（哈希类型）。这些数据类型都支持push/pop、add/remove及取交集并集和差集及更丰富的操作，而且这些操作都是原子性的。在此基础上，redis支持各种不同方式的排序。与memcached一样，为了保证效率，数据都是缓存在内存中。区别的是redis会周期性的把更新的数据写入磁盘或者把修改操作写入追加的记录文件，并且在此基础上实现了master-slave(主从)同步。

MySQL常用操作

发表于 2015-10-12 | 分类于 mysql | 阅读次数

设置指定的自增主键
alter table table_name AUTO_INCREMENT=1000;

修改字段名称与类型：
alter table table_name change old_field new_filed new_filed_type;

清空表：
truncate table table_name;

删除表字段：
alter table table_name drop column field;

增加表字段：

alter table table_name add filed type ...;
复制表结构：
create table table1 like table2;

从一个数据表导入数据到另一个数据表：
insert into table(field) select field form table1;
